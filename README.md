# GourmetList - Smart Shopping & Chef Bot

Built by [Teda.dev](https://teda.dev), the AI app builder for everyday problems.

GourmetList is a modern, privacy-focused shopping list application with a built-in AI Chef. It runs entirely in your browser using WebGPU technology, meaning your data and queries never leave your device.

## Features

- **Smart List Management**: Easily add, check off, and delete items.
- **AI Chef Bot**: A local LLM (Qwen2.5-1.5B) that lives in the sidebar. Ask it for recipes based on your specific list items.
- **Local Persistence**: Your list and chat history are saved automatically.
- **Modern Design**: A fluid, responsive interface with beautiful animations.

## Technology Stack

- HTML5, CSS3 (Tailwind CSS)
- jQuery 3.7.x
- WebLLM (Client-side AI)

## Usage

1. Open `index.html` to view the landing page.
2. Click "Launch App" to go to the main application.
3. Wait for the AI model to download (cached for subsequent visits).
4. Add ingredients and ask the Chef Bot for ideas!
